🔒 隐私政策

简单说：我们非常重视你的隐私。MyLuck 是一个纯静态网站，所有测试都在你的浏览器里运行，我们不会偷偷收集你的任何个人信息。下面是详细说明，但如果你时间有限，知道"我们不收集你的数据"就够了 😊

我们收集什么？

简短回答：几乎什么都不收集。

  • 我们不需要你注册或登录
  • 我们不收集你的姓名、邮箱、电话等个人信息
  • 所有测试（运气测试、MBTI、色彩测试等）完全在你的浏览器本地运行
  • 如果你选择「上榜」，你输入的昵称和测试分数会被提交到我们的排行榜服务器（Supabase），以便所有用户都能看到
  • 留言板发布的内容也会储存在 Supabase 服务器上，公开可见
  • 你的语言偏好保存在浏览器的 localStorage 中，只存在你自己的设备上

第三方服务

为了让网站正常运行和持续改进，我们使用了一些第三方服务，它们可能有自己的隐私政策：

  • GitHub Pages — 我们的网站托管在这里。GitHub 可能会记录访问者的 IP 地址用于安全目的。GitHub 隐私声明
  • 匿名统计 — 我们使用隐私友好的分析工具来了解哪些页面更受欢迎，以便持续改进体验。这些工具不会收集你的个人身份信息，不使用 Cookie 进行跟踪。
  • 广告服务 — 如果我们接入了广告，广告提供商可能使用 Cookie 来展示相关广告。你可以通过浏览器设置管理 Cookie。
  • Supabase — 我们使用 Supabase 存储排行榜数据和留言板内容。仅储存你主动提交的昵称、分数和留言内容，不关联任何个人身份信息。Supabase 隐私政策
  • Cloudflare Turnstile — 为防止机器人滥用，排行榜和留言板提交时使用 Cloudflare 人机验证。Cloudflare 可能在验证过程中处理有限的设备信息。Cloudflare 隐私政策

Cookie

MyLuck 本身不使用 Cookie。但第三方服务（如广告）可能会使用。你的浏览器设置可以帮你管理这些。

适合所有年龄

我们的内容面向所有年龄段的用户，不包含任何不适合未成年人的内容。我们也不会主动向未成年人收集任何信息。

有疑问？

如果你对隐私有任何疑问或担忧，欢迎通过留言板联系我们，我们会尽快回复。
