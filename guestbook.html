⚠️ Giscus 配置指南（上线前请完成）

  1. 前往 GitHub 创建你的仓库（如果还没有）
  2. 在仓库 Settings → Features 中启用 Discussions
  3. 前往 giscus.app 配置你的仓库
  4. 将生成的配置填入上方 script 标签的 data-repo、data-repo-id、data-category、data-category-id 属性
  5. 上线前删除此说明区域

Giscus 优势：基于 GitHub Discussions，自带用户认证（防止匿名刷屏），支持 Markdown 和表情回复，完全免费。
